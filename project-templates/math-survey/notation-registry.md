# 全局符号注册表

所有正文中使用的数学符号在此注册。一个符号在全项目中只能有一个含义。多终端只追加。

## 已固定符号

| 符号 | 含义 | 适用章节 | 首次定义 |
|------|------|---------|---------|
| $B_n$ | Bernoulli 数（$B_1 = -1/2$ 约定） | 全局 | §1.1 |
| $B_n(x)$ | Bernoulli 多项式 | 全局 | §9.1 |
| $(\text{C}, k)$ | Cesàro 求和法，阶 $k$ | 全局 | §4.1 |
| $(\text{A})$ | Abel 求和法 | 全局 | §4.2 |
| $(\text{B})$ | Borel 求和法 | 全局 | §5.1 |
| $(\text{E}, q)$ | Euler 求和法，参数 $q$ | 全局 | §4.3 |
| $\mathfrak{R}$ | Ramanujan 求和 | 全局 | §6.1 |
| $\zeta(s)$ | Riemann zeta 函数 | 全局 | §7.1 |
| $\zeta(s, a)$ | Hurwitz zeta 函数 | 全局 | §7.3 |
| $\mathbb{K}[[x]]$ | 域 $\mathbb{K}$ 上的形式幂级数环 | 全局 | §9.1 |
| $\Delta$ | 差分算子 $\Delta f(x) = f(x+1) - f(x)$ | 第9-11章 | §9.3 |
| $q$ | 模参数 $q = e^{2\pi i \tau}$（除非声明使用 nome） | 第12-14章 | §12.2 |

## 需消歧符号

以下符号在不同章节有不同含义，每次使用必须在本章首次出现时消歧（如上标 `$E_n^{\text{Euler}}$` 或脚注声明）。

追加新含义时直接在对应符号下方追加一行，无需改表格结构。

- **$E_n$**
  - Euler 数（第9,11章）
  - Eisenstein 级数 $E_{2k}$（第14章）
